# GEOInsightLab Web

Aplicação científica interativa associada à investigação doutoral
**Ciência de Dados na Literacia Espacial**.

## Estado desta versão

Esta é a Fase 1 estável do novo repositório:

- navegação;
- identidade visual;
- carregamento e validação dos dados;
- página inicial;
- módulo Data Exploration;
- mapa, filtros, gráficos, estatística descritiva e exportação.

## Ficheiro principal no Streamlit Cloud

```text
streamlit_app.py
```

## Executar localmente

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run streamlit_app.py
```
